# Auto Hide Keyboard (SillyTavern Android Fix)

This extension automatically hides the Android keyboard after sending a message in SillyTavern.

## Why this exists
On Android (Chrome / Samsung Internet), SillyTavern keeps the input focused after sending a message, leaving the keyboard open and taking up screen space.

## Features
- Hides keyboard after Send button press
- Hides keyboard after Enter key send
- Prevents Android Chrome from immediately refocusing input
- Works with SillyTavern 1.18.0
